## How to Use These Tables
1. When starting a session, roll for Campaign Hook and Current Location
2. Roll for Local Problem to establish immediate conflict
3. Generate 2-3 Important NPCs using all three subtables
4. Add Location Features for detail
5. When action slows, roll for Encounter Type
6. Use Plot Twist when needed to raise stakes
7. Monster Motivation helps create meaningful combat
8. Roll for Treasure Type and Complications as needed

### Quick Scene Builder:
1. Roll Location Features
2. Roll Encounter Type
3. Roll one NPC
4. Roll one Complication

### Emergency Plot Generator:
1. Roll Campaign Hook
2. Roll Local Problem
3. Roll Plot Twist
4. Roll two NPCs

Remember: Let dice rolls inspire rather than restrict your storytelling. Feel free to modify or interpret results to better fit your narrative.
